# Yakshapriya
It is a Web development project.

->It is  a platform for yakshagana Priyaâ€™s(lovers).

->This platform gives user to view the places and things where the program is going to be held.

->The organiser of the program can also upload details about their function.

->User can view all organised function details. 

->User can also comment on the program and can give feedback.





##--INPUT OF THE PROJECT--##
   1. Organiser details
   2. Details of artists
   3. Ticket confirmation
   4. Payment.




      
#--OUTPUT OF THE PROJECT--##
   1. Confirmed tickets or seats
   2. Available seats
   3. Seat reservation details
   4. Bill generation for booked seat




#---LIMITATION OF THE PROJECT--#

-->No multi-language support. 

-->Credit debit and net banking payment procedure cannot be done.

-->User cannot cancel their seat after the payment. 

-->Live tracking of the place is not available. 

-->Login through facebook,twitter etc are not possible.
